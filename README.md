## API Endpoint

| Endpoint        | Method | Description                            |
| --------------- | ------ | ---------------------------------------|
| /tickets        | GET    | Show all tickets                       |
| /tickets/:id    | GET    | Show one ticket with ticket number     |
| /tickets/create | POST   | Create one ticket                      |
